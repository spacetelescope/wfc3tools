.. _wf3ir:

wf3ir
=====

.. automodapi:: wfc3tools.wf3ir

Command Line Options for the ``wf3ir`` C Executable
---------------------------------------------------

The wf3ir function can also be called directly from the OS command line:

.. code-block:: shell

    wf32ir.e input output [-options]

Input may be a single filename, and the options include:

* -v: verbose
* -t: print time stamps
